## ‚úÖ Yechim: Shrift O'zgartirish

### Qilingan Ish:
**File:** `app/globals.css`

**O'zgarish:**
```css
/* Oldin */
font-family: var(--font-sans), system-ui, sans-serif;

/* Keyin */
font-family: 'Space Grotesk', 'Inter', 'Segoe UI', 'Roboto', 'Arial', sans-serif;
```

### Nega Bu Shriftlar?

1. **Space Grotesk** - Loyiha asosiy shrifti, o'zbek harflarini qo'llab-quvvatlaydi
2. **Inter** - Zamonaviy, barcha Unicode harflarni qo'llab-quvvatlaydi
3. **Segoe UI** - Windows standart shrifti, yaxshi qo'llab-quvvatlash
4. **Roboto** - Google shrifti, keng qo'llab-quvvatlash
5. **Arial** - Fallback, hamma joyda mavjud
